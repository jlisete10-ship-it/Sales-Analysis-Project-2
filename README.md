# Sales-Analysis-Project-2
This project analyses sales data using Python, SQL, and Power BI.
The goal is to clean the data, extract insights, and identify key business risks and opportunities.

---

1. Data Cleaning (Python)

* Handled missing values in customer and product columns
* Converted incorrect data types in the value column
* Replaced invalid values (e.g., "error") with numeric values
* Created a new column: 'total_spent'

---

2. Data Analysis (Python)

* Identified the top customer by total spending
* Found the most profitable product
* Calculated customers above average spending
* Computed percentage contribution of each customer

---

3. SQL Analysis

* Total sales per customer
* Customers above average using subqueries

---

4. Dashboard (Power BI)

* KPI: Total Sales
* Sales by Customer
* Sales by Product
* Percentage contribution

---

5. Key Insights

* Rui is the most important customer, contributing the majority of revenue
* There is a high business risk** due to dependency on a single customer
* Monitor is the most impactful product due to its high value

---

Notes

Some records contain missing or unknown values, which were handled during data cleaning.

---

Tools Used

* Python (Pandas)
* SQL
* Power BI
